جلسه اول : tailwin css چیست؟

شروع یادگیری فرانت مودرن!

Tailwind CSS یک فریم‌ورک Utility-first است که به جای نوشتن استایل‌های جداگانه، به تو کلاس‌های کوچک و آماده می‌دهد تا ظاهر عناصر را مستقیماً داخل HTML بسازی. با Tailwind می‌توانی بدون نوشتن حتی یک خط CSS، خیلی سریع رابط کاربری‌های تمیز، ریسپانسیو و قابل‌سفارشی‌سازی بسازی و همه‌چیز را دقیقاً هم‌زمان با کدنویسی طراحی کنی.

Install Tailwind CSS (v3.4)

در مرحله اول توجه داشته باشید که حتما node.js در سیستم شما نصب شده باشد. برای نصب Tailwind CSS در یک پروژه جدید، مراحل زیر را انجام دهید:

1. ایجاد پروژه و نصب پکیج‌ها Tailwind را به همراه PostCSS و Autoprefixer نصب کنید: 

// نصب تیلویند (کامند)
        npm install -D tailwindcss@3

سپس فایل‌های پیکربندی را ایجاد کنید: 

//  پیکربندی (کامند)
        npx tailwindcss init

2. تنظیم مسیرهای Template فایل tailwind.config.js را باز کنید و مسیر فایل‌هایی که قرار است کلاس‌های Tailwind در آن‌ها استفاده شود را مشخص کنید: 

// tailwind.config.js
module.exports = {
  content: [
    "./index.html",
    "./src/**/*.{js,html}"
    ],
  theme: {
    extend: {},
  },
  plugins: [],
}

3. اضافه کردن Directives محتوای فایل CSS اصلی را به شکل زیر تنظیم کنید: 

// input.css
@tailwind base;
@tailwind components;
@tailwind utilities;

سپس tailwind را ایجاد کنید: 

//package.json
        {
        "devDependencies": {
          "tailwindcss": "^3.4.18"
          
        },
//برای ران کردن پروژه
        "scripts": {
          "tailwind": "tailwindcss -i ./css/input.css -o ./css/main.css --watch"
        }
      }

حال میتوانین با این کامند پروژه را ران کنید 

//cmd
nmp run tailwind

فاصله ها در tailwindcss

انواع فاصله ها:

  • margin (حاشیه بیرونی)
  • padding (فضای درونی)
  • gap (فاصله بین المان‌های شبکه‌ای مثل Flexbox و Grid)
margin (حاشیه بیرونی):

کلاس کاربرد مثال
m-{size} حاشیه کلی در تمام جهات m-4
mx-{size} حاشیه افقی (چپ و راست) mx-2
my-{size} حاشیه عمودی (بالا و پایین) my-3
mt-{size} حاشیه بالا mt-6
mr-{size} حاشیه راست mr-1
mb-{size} حاشیه پایین mb-5
ml-{size} حاشیه چپ ml-0
padding (فضای درونی):
کلاس کاربرد مثال
p-{size} فضای داخلی در تمام جهات p-4
px-{size} فضای داخلی افقی (چپ و راست) px-2
py-{size} فضای داخلی عمودی (بالا و پایین) py-3
pt-{size} فضای داخلی بالا pt-6
pr-{size} فضای داخلی راست pr-1
pb-{size} فضای داخلی پایین pb-5
pl-{size} فضای داخلی چپ pl-0
راهنمای سایز ها :
مقدار معادل تقریبی (rem) توضیح
0 0 بدون فاصله
1 0.25rem (4px) فاصله کوچک
2 0.5rem (8px) فاصله متوسط
4 1rem (16px) فاصله معمول
8 2rem (32px) فاصله بزرگ
px 1px ۱ پیکسل
اما gap به همراه flex & Grid میاید (جلسه 2)
